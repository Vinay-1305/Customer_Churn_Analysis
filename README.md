Customer Churn & Retention Analysis

Problem Statement:
The objective of this project is to analyze customer churn behavior
and identify key factors contributing to customer attrition,
and provide data-driven business recommendations.

Tools Used:
- Excel (Data Cleaning, Pivot Tables)
- SQL (optional â€“ future extension)
- Power BI (optional)

Analysis Steps:
1. Data cleaning and validation
2. Tenure vs Churn analysis
3. Monthly Charges vs Churn analysis
4. Contract Type vs Churn analysis
5. Identification of churn drivers
6. Business recommendations

Key Insights:
- Churn is highest in the first 12 months (~48%)
- Month-to-month contracts show highest churn (~43%)
- High monthly charges increase churn risk

Business Recommendations:
- Improve first-year onboarding
- Incentivize long-term contracts
- Retain high-value customers with loyalty programs
